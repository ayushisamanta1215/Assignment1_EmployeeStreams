# Assignment 3 - Banking Transaction Analyzer

## Features
- Filter debit transactions
- Find high-value transactions (> ₹50,000)
- Calculate total credited amount
- Group transactions by type
- Find latest transaction
- Sort transactions by amount
- Use parallel stream processing

## Concepts Used
- Streams
- Parallel Streams
- filter()
- mapToDouble()
- groupingBy()
- Comparator
- max()
