# Assignment 5 - Mini Employee Management Backend

## Objective
Build backend modules using:
- Java 8
- Streams
- Optional
- REST API Design
- Microservices Thinking

---

## Modules

### 1. Employee Service
- Add Employee
- Update Employee
- Get Employee
- Delete Employee

### 2. Payroll Service
- Calculate Salary
- Calculate Tax
- Generate Payslip

### 3. Notification Service
- Send Email
- Send SMS

---

## Features Implemented

- Stream API for filtering employees
- Optional for null handling
- REST API design
- Service separation
- Structured API request/response

---

## Example APIs

POST /employees  
GET /employees  
PUT /employees/{id}  
DELETE /employees/{id}
