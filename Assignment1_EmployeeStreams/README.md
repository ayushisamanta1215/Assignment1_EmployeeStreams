# Assignment1_EmployeeStreams

## Objective
This project demonstrates the use of Java Stream API with employee data processing.

## Concepts Used
- Streams
- filter()
- map()
- sorted()
- collect()
- Comparator
- groupingBy()
- max()

## Problem Statement
Create an Employee class with:
- id
- name
- department
- salary
- activeStatus

## Operations Performed
1. Fetch employees with salary > 60,000
2. Convert employee names to uppercase
3. Sort employees by salary (descending)
4. Count active employees
5. Group employees by department
6. Find highest salary employee
7. Find second highest salary employee

## Technologies Used
- Java 8+
- Stream API

## How to Run
1. Clone the repository
2. Compile the Java file
3. Run the Main class
